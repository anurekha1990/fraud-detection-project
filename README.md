# fraud-detection-project
Developed a fraud detection system using Logistic Regression and Random Forest on imbalanced data. Applied SMOTE for class balancing and improved model performance using precision, recall, and F1-score.
## Overview
This project aims to detect fraudulent transactions using machine learning techniques on an imbalanced dataset.

## Features
- Data preprocessing and scaling
- Handling imbalanced data using SMOTE
- Model training using Logistic Regression and Random Forest
- Performance evaluation using precision, recall, and F1-score

## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

## Results
- Random Forest performed better in detecting fraudulent transactions
- Improved recall after applying SMOTE

## Dataset
- Credit Card Fraud Detection Dataset (Kaggle)
